# VidaSalud

VidaSalud es una aplicación para Android diseña para promover un estilo de vida saludable, permitiendo a los usuarios interactuar en una comunidad, compartir y recibir consejos.

## Integrantes

-Miguel Tropa
-Daniel Vargas


## Funcionalidades Implementadas

- **Navegación con Jetpack Compose:** La aplicación utiliza un grafo de navegación para moverse entre las diferentes pantallas.
- **Comunidad:** Permite a los usuarios interactuar en una comunidad, donde pueden ver publicaciones y consejos de otros usuarios.
- **Interfaz de Usuario Moderna:** La interfaz está construida con Jetpack Compose, siguiendo las últimas prácticas de desarrollo de Android.

## Pasos para Ejecutar el Proyecto

1.  **Clonar el Repositorio:**
    ```
    git clone https://github.com/Danvargast/VidaSalud.git
    ```
2.  **Abrir en Android Studio:**
    - Abre Android Studio.
    - Selecciona "Open an existing Android Studio project".
    - Navega hasta la carpeta donde clonaste el repositorio y selecciónala.
3.  **Sincronizar Gradle:**
    - Android Studio debería sincronizar el proyecto automáticamente. Si no lo hace, haz clic en el botón "Sync Project with Gradle Files" (el elefante de Gradle) en la barra de herramientas.
4.  **Ejecutar la Aplicación:**
    - Selecciona un emulador o un dispositivo físico.
    - Haz clic en el botón "Run 'app'" (el triángulo verde) en la barra de herramientas.
